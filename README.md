# elpe.stock.valuation
DCF valuation and financial analysis of HELLENiQ ENERGY (ELPE), benchmarked against Motor Oil (MOH). Covers historical ratios (2021–2025) — liquidity, cash conversion cycle, capital structure, Altman Z-score — and a 2026–2030 unlevered FCF forecast with WACC-based enterprise/equity valuation, implied share price, and sensitivity analysis.
